# customer_behavior_analysis
data analytics project showcasing customer behavior analysis using python, sql and power bi.
📊 Data Analytics Project – End-to-End Analysis
📌 Overview

This project demonstrates an end-to-end data analytics workflow, including data loading, cleaning, exploratory data analysis (EDA), SQL-based querying, and dashboard creation. The goal is to extract meaningful insights and support data-driven decision-making.

🛠️ Tools & Technologies

Python (Pandas, NumPy, Matplotlib, Seaborn)

SQL (MySQL / PostgreSQL)

Power BI / Tableau (for dashboard)

Excel (for initial data review)

📂 Dataset
Type: Structured dataset (CSV/Excel)

Size: [3900 × 18]

🔍 Project Steps
1. Data Loading

Imported dataset using Pandas

Checked structure, datatypes, and missing values

2. Data Cleaning

Handled missing values using imputation/removal

Removed duplicates

Standardized column names and formats

Converted data types where necessary

3. Exploratory Data Analysis (EDA)

Identified trends, patterns, and anomalies

Used visualizations (bar charts, histograms, heatmaps)

Performed correlation analysis

4. SQL Analysis

Wrote queries to extract business insights

Used joins, aggregations, and filtering

Created summary tables for reporting

5. Dashboard Creation

Built interactive dashboard using Power BI/Tableau

Visualized KPIs and trends

Enabled filtering for better user interaction

📊 Key Results

Identified key factors influencing the dataset trends

Improved data quality by cleaning and preprocessing

Generated actionable insights using SQL queries

Created a dashboard for easy business interpretation
